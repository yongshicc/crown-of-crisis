# Crown of Crisis — GitHub Light Build

This is the compressed GitHub-friendly Web Game build.

## How to run
Open `index.html` in a browser.

## How to publish on GitHub Pages
1. Upload the full folder contents to a GitHub repository.
2. Keep the `assets/` folder next to `index.html`.
3. Enable GitHub Pages for the repository.
4. Set the Pages source to the branch/folder containing `index.html`.

This build uses external compressed WebP assets instead of embedding all images as base64 inside the HTML.
